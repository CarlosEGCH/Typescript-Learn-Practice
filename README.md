# TypeScript Learn & Practice

A personal repository for learning and practicing TypeScript — from basics to advanced patterns.

## Goals

- Understand TypeScript's type system progressively
- Practice with real examples and exercises
- Build a reference I can come back to

## Structure

| Folder | Topics |
|---|---|
| `src/01-basics/` | Primitive types, interfaces, functions |
| `src/02-intermediate/` | Generics, enums, type narrowing |
| `src/03-advanced/` | Utility types, decorators, conditional types |
| `src/04-exercises/` | Challenges with solutions |
| `notes/` | Cheatsheet and reference notes |

## Setup

```bash
npm install
npm run dev       # run a file with ts-node
npm run check     # type-check without emitting
```
